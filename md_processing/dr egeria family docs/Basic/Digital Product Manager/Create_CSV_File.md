# Create CSV File
>	Create a CSV File asset

## Display Name
>	**Input Required**: True

>	**Description**: Name of the CSV File

>	**Alternative Labels**: File Name


## Description
>	**Input Required**: False

>	**Description**: Description of the CSV file.


## File Path
>	**Input Required**: False

>	**Description**: Path to the file including file name. File system name is pre-pended if set.

>	**Alternative Labels**: File Path Name


## File Encoding
>	**Input Required**: False

>	**Description**: Encoding of the file.

>	**Default Value**: UTF-8


## File Extension
>	**Input Required**: False

>	**Description**: Typically CSV.

>	**Default Value**: csv


## File System Name
>	**Input Required**: False

>	**Description**: Optional  file system name to be prepended in front of the path name.


## User Defined Status
>	**Input Required**: False

>	**Description**: Only valid if Product Status is set to OTHER. User defined & managed status values.


## Category
>	**Input Required**: False

>	**Description**: A user specified category name that can be used for example, to define product types or agreement types.

>	**Alternative Labels**: Category Name


## Version Identifier
>	**Input Required**: False

>	**Description**: Published product version identifier.

>	**Default Value**: 1.0
