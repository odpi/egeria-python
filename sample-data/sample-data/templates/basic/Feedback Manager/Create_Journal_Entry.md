# Create Journal Entry
>	Allows a sequence of notes to be attached to an associated element.
>	**Alternative Names**: Note Log; Journal

# Required

## Journal Name
>	**Input Required**: True

>	**Description**: Name of the  NoteLog or Journal.

>	**Alternative Labels**: Name; Title


## Note Entry
>	**Input Required**: True

>	**Description**: The text of the journal entry.

>	**Alternative Labels**: Note; Journal Entry
